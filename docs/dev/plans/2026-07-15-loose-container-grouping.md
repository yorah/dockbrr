# Loose (Docker-auto-named) container grouping Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Group standalone containers that Docker auto-named (`adjective_surname`) into one collapsed-by-default "Loose" section in the sidebar and dashboard, so orphan noise stops burying real projects.

**Architecture:** A Go classifier (`dockername`) detects Docker-assigned names against moby's frozen word lists. Discovery stamps a project-level `auto_named` flag, which flows store -> API -> frontend types. The sidebar and dashboard render `auto_named` projects inside a collapsible "Loose (N)" group, collapsed by default. No behavior change beyond presentation: loose containers stay discovered and tracked.

**Tech Stack:** Go 1.26 (CGO-free), SQLite (migration), React + TypeScript + TanStack Table, Vitest + Testing Library + MSW.

## Global Constraints

- Static binary invariant: `CGO_ENABLED=0 go build ./...` must stay green; no new cgo deps.
- Classifier gated to standalone: `auto_named = kind == "standalone" && dockername.IsDockerAssigned(name)`. A compose project is never loose.
- Collapse-by-default is the only "hide" mechanism. Do NOT add a Filters-bar toggle.
- Label text is exactly `Loose` (group header renders `Loose (N)`).
- No em-dashes in code comments or docs.
- TS typecheck via `./node_modules/.bin/tsc -b --noEmit` (NOT `npx tsc`); `npm run build` is the backstop.
- Verify command (backend): `CGO_ENABLED=0 go build ./... && go vet ./... && go test ./...`
- Verify command (frontend, from `web/`): `npm test`

```go
// Package dockername detects container names that Docker auto-assigned from its
// frozen namesgenerator word lists ("adjective_surname"). Such a name means the
// user never explicitly named the container, so it is likely disposable.
package dockername

import "strings"

// IsDockerAssigned reports whether name matches Docker's auto-generated
// "adjective_surname" form, optionally followed by a single collision digit
// (e.g. "focused_turing3"). Detection is exact against the frozen word lists:
// a name matches only when both halves are Docker vocabulary.
func IsDockerAssigned(name string) bool {
	// Strip a single optional trailing digit: GetRandomName appends 0..9 on a
	// name collision. No adjective or surname ends in a digit, so this only
	// removes the collision suffix.
	if n := len(name); n > 0 {
		if c := name[n-1]; c >= '0' && c <= '9' {
			name = name[:n-1]
		}
	}
	// Adjectives and surnames contain no underscore, so the first underscore is
	// the separator.
	i := strings.IndexByte(name, '_')
	if i <= 0 || i >= len(name)-1 {
		return false
	}
	if _, ok := adjectives[name[:i]]; !ok {
		return false
	}
	_, ok := surnames[name[i+1:]]
	return ok
}
```

```tsx
export function DashboardTable({
  rows,
  onReview,
  updatesByService,
  onApplied,
  onChangelog,
  groupLoose = false,
  looseDefaultOpen = false,
}: DashboardTableProps) {
  const [collapsed, setCollapsed] = useState<Set<number>>(() => new Set());
  const [composeProject, setComposeProject] = useState<Project | null>(null);
  const [looseOpen, setLooseOpen] = useState(looseDefaultOpen);

  // Auto-expand under an active filter, re-collapse when it clears.
  useEffect(() => setLooseOpen(looseDefaultOpen), [looseDefaultOpen]);

  const visibleRows = useMemo(() => {
    const shown = (r: Row) => r.kind !== "service" || !collapsed.has(r.project.id);
    if (!groupLoose) {
      return rows.filter(shown);
    }
    const normal = rows.filter((r) => r.kind !== "loose" && !r.project.auto_named);
    const loose = rows.filter((r) => r.kind !== "loose" && r.project.auto_named);
    const looseCount = loose.filter((r) => r.kind === "project").length;
    const out: Row[] = normal.filter(shown);
    if (looseCount > 0) {
      out.push({ kind: "loose", count: looseCount });
      if (looseOpen) out.push(...loose.filter(shown));
    }
    return out;
  }, [rows, collapsed, groupLoose, looseOpen]);
```
